# Security Policy

## Supported Versions

We actively support and patch the latest major version of `envrepair`.

| Version | Supported          |
| ------- | ------------------ |
| v0.x    | :white_check_mark: |

## Reporting a Vulnerability

We take the security of `envrepair` seriously. If you find a security vulnerability, please do **not** open a public issue. Instead, report it privately to ensure the safety of our users.

### Reporting Channels

- **GitHub Private Vulnerability Reporting**: You can report security vulnerabilities directly on GitHub via our repository's security tab using the "Report a vulnerability" button.
- **Direct Contact**: Alternatively, you can email the maintainer at `avenolazo@gmail.com` with the details.

Please include:

- A detailed description of the vulnerability.
- Steps to reproduce (a Proof of Concept).
- Any potential impact or exploit scenarios.

We will acknowledge your report within 48 hours and work with you to coordinate a security fix and disclosure.
